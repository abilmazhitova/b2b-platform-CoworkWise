# CoworkWise - Coworking Location Intelligence Platform

A modern B2B analytics platform for analyzing and selecting optimal coworking space locations in Almaty, Kazakhstan.

## Features

- **Interactive Map**: Real-time heatmap visualization of foot traffic and coworking locations
- **Location Analysis**: Compare districts, view AI-powered recommendations, and forecast demand
- **Admin Dashboard**: Manage users, data layers, and system settings
- **User Profiles**: Track activity and save favorite locations

## Tech Stack

- **Frontend**: Next.js 15 with React 19
- **UI Library**: Tailwind CSS v4 + shadcn/ui
- **Maps**: React-Leaflet + Leaflet.heat
- **Charts**: Recharts
- **State Management**: Zustand
- **Authentication**: JWT (ready for backend integration)
